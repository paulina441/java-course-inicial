HTML

  1. Etiqueta:

    2. Bloque de construcción.

  2. Atributo

    3. Información adicional para modificar el comportamiento o propiedades.

  3. Valor

    4. Dato específico que se asigna a un atributo.

  • Encabezado (HEADER)

    • Introducción, navegación principal

  • Contenido principal (MAIN)

    • Contenido principal de la página web, solo se usa una vez por página.

  • Pie de página (FOOTER)

    • Parte final de la página que contiene información como contacto, derechos de autor, enlaces o redes sociales.

CSS

  1. Inline

    2. CSS en linea, se esbribe directacmente en el HTML.

  2. Interno

    3. CSS interno, se define dentro del elemento style en el head.

  3. Externo

    4. CSS externo, es un archivo .css separado y vinculado al HTML.

  • Padding

    • Espacio entre el contenido y el border.

  • Border

    • Linea que rodea el padding y el contenido.

  • Margin

    • Distancia entre un elemento y otro.
